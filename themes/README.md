# THRM 自定义主题

这个文件夹用于存放 **自定义界面主题**。应用启动时会扫描这里的每个子文件夹，
把它们识别成「系统设置 → 界面主题」里可选的主题。

> 内置的「浅色 / 深色 / 跟随系统」三种基础主题不在这里，它们由程序自带；
> 这里放的是**可自由编辑**的自定义主题。官方示例 `thrm`（冰川蓝）就是其中之一。

## 目录在哪

- 安装目录下的 `themes\` 文件夹（一般就是本文件夹）。
- 如果安装目录不可写（例如装在 `C:\Program Files\`），应用会改用用户目录：
  `C:\Users\你的用户名\.thrm\themes\`。两个位置的主题都会被识别；同名（同 id）时用户目录优先。

可以在应用的「界面主题」设置项旁点击「打开主题文件夹」按钮，直接定位到这里。

## 一个主题 = 一个文件夹

每个主题是一个子文件夹，里面包含两个文件：

```
themes/
└─ thrm/                ← 文件夹名建议与 theme.json 里的 id 一致
   ├─ theme.json        ← 主题清单（名字、id、基于浅色还是深色）
   └─ theme.css         ← 主题样式（改这里就能改外观）
```

### theme.json 字段

| 字段          | 必填 | 说明 |
|---------------|------|------|
| `id`          | 是   | 唯一标识，只能用小写字母/数字/`-`/`_`。会作为 `<html data-theme="id">` 注入页面。 |
| `name`        | 是   | 下拉菜单里显示的名字。 |
| `base`        | 否   | `light` 或 `dark`，本主题基于哪种底色，默认 `light`。 |
| `author`      | 否   | 作者。 |
| `version`     | 否   | 版本号。 |
| `description` | 否   | 简介。 |

### theme.css 写法（重要）

所有选择器都要以 `html[data-theme="你的id"]` 开头，样式才只在该主题被选中时生效：

```css
/* 把主色改成紫色 */
html[data-theme="mytheme"] {
  --primary: #7c5cff;
  --ring: #7c5cff;
  --chart-primary: #7c5cff;
}
```

`thrm/theme.css` 里有非常详细的中文注释，演示了从「只换颜色」到「毛玻璃 / 网格背景 /
扫描动画」等各种程度的定制，是最好的参考与起点。

## 怎么做自己的主题

1. 把 `thrm` 整个文件夹复制一份，重命名（例如 `mytheme`）。
2. 编辑 `mytheme/theme.json`：把 `id` 改成 `mytheme`、`name` 改成你喜欢的名字。
3. 编辑 `mytheme/theme.css`：把里面所有 `html[data-theme="thrm"]` 替换成
   `html[data-theme="mytheme"]`，然后按注释修改颜色/样式。
4. 回到应用「系统设置 → 界面主题」，选中你的主题即可。修改 CSS 后，重新选择一次
   主题或重启应用即可看到更新。

## 可用的「界面钩子」类名

界面里预留了一批以 `glacier-` 为前缀的类名，你的主题可以直接选中它们做深度定制
（详见 `thrm/theme.css` 顶部的列表与注释）。另外有一个特殊类：

- `.theme-thrm-only`：默认全局隐藏，只有在选中本主题（或你复制出的主题）时才显示，
  用于放置「仅该主题才出现」的装饰元素。
